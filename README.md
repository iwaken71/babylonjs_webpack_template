# babylonjs_webpack_template
# 背景
Babylon.js + webpackのサンプルを目指しました。

# Setup

clone
``` 
git git@github.com:iwaken71/babylonjs_webpack_template.git
cd babylonjs_webpack_template
```

SetUp

```
npm install
```

Dev実行

```
npm run dev
```

Build
./dist以下にbundleファイルができる
```
npm run build
```

dist以下のファイルをホスティングするとWebページとして動作する。
